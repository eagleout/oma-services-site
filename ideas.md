# Brainstorming Design - Nettoyage Premium & Services Humains

## Contexte
Site web pour une société de nettoyage française avec deux pôles : particuliers (aide à domicile, seniors, handicap) et professionnels (bureaux, commerces). Le site doit inspirer immédiatement confiance, sérieux, bienveillance, propreté et qualité.

---

## Approche 1 : Minimalisme Institutionnel Chaleureux
**Probabilité : 0.08**

### Design Movement
Néo-institutionnel moderne : fusion entre l'autorité rassurante d'une institution publique de confiance et la chaleur d'une entreprise humaine locale.

### Core Principles
1. **Épuration radicale** : Chaque élément a une fonction. Espace blanc généreux comme symbole de clarté et de sérénité.
2. **Hiérarchie typographique forte** : Contraste marqué entre titres (serif élégant) et corps (sans-serif épuré) pour guider l'attention.
3. **Confiance par la structure** : Grille régulière, alignements impeccables, sections bien délimitées = rigueur opérationnelle.
4. **Humanité par les détails** : Micro-interactions fluides, images authentiques (pas de clichés), témoignages vrais.

### Color Philosophy
- **Palette primaire** : Blanc pur (fond), bleu-gris profond (confiance, sérieux) #2C3E50, accent vert doux (bien-être, nature) #5FB878
- **Reasoning** : Le bleu-gris évoque la confiance institutionnelle ; le vert symbolise le bien-être et la propreté naturelle. Le blanc crée l'espace respirant.
- **Emotional intent** : Rassurant sans être froid, professionnel sans être distant.

### Layout Paradigm
- **Hero asymétrique** : Texte/CTA à gauche, image/visual à droite (déséquilibre intentionnel = dynamisme)
- **Sections alternées** : Texte + image alternent de côté (gauche/droite) pour éviter la monotonie
- **Grille 3 colonnes** pour les services, mais avec espacement variable
- **Dividers organiques** : Transitions douces entre sections (pas de coupures brutales)

### Signature Elements
1. **Ligne verticale fine** : Accent graphique qui sépare les sections, symbolise la rigueur
2. **Icônes géométriques épurées** : Formes simples, lignes fines, palette limitée
3. **Cartes de service avec coin coupé** : Détail distinctif qui rompt la rectitude parfaite

### Interaction Philosophy
- Transitions fluides et prévisibles (0.3s ease-out)
- Boutons CTA avec underline animé (pas de fond plein)
- Hover subtle : légère élévation + changement de couleur douce
- Scroll-triggered animations : sections qui se révèlent progressivement

### Animation
- **Entrance** : Fade-in + slide-up léger (200ms)
- **Hover** : Underline qui s'étire (300ms), légère translation (2px)
- **Scroll** : Parallax subtil sur les images (vitesse réduite)
- **CTA** : Pulse léger au chargement de la page

### Typography System
- **Display** : Playfair Display (serif, 48-64px) pour les titres principaux = élégance
- **Heading** : Poppins Bold (sans-serif, 28-36px) pour les sous-titres = clarté
- **Body** : Inter Regular (sans-serif, 16px) pour le texte courant = lisibilité
- **Accent** : Poppins SemiBold pour les labels et CTAs = distinction
- **Hierarchy** : Contraste de taille 2:1 minimum entre niveaux

## Approche 2 : Modernisme Organique et Humain
**Probabilité : 0.07**

### Design Movement
Modernisme organique : formes courbes, asymétrie naturelle, palette terreuse, mais avec une rigueur technologique sous-jacente.

### Core Principles
1. **Formes organiques** : Courbes, arrondis généreux, shapes irrégulières qui évoquent la nature et l'humanité
2. **Asymétrie intentionnelle** : Compositions non-centrées, décalages, qui créent du mouvement
3. **Chaleur par la couleur** : Palette terreuse (beige, ocre, vert sage) pour l'accessibilité émotionnelle
4. **Technologie invisible** : Fonctionnalité parfaite mais interface qui semble naturelle, pas "corporate"

### Color Philosophy
- **Palette primaire** : Beige chaud #F5F1E8 (fond), terracotta #C85A54 (énergie, action), vert sage #7A9B7F (sérénité)
- **Accents** : Ocre #D4A574 pour les highlights, gris chaud #8B8680 pour le texte
- **Reasoning** : Palette terreuse = proximité, chaleur humaine. Terracotta = action et énergie bienveillante. Vert sage = nature, confiance.
- **Emotional intent** : Accueillant, rassurant, naturel, mais professionnel.

### Layout Paradigm
- **Grille fluide** : Colonnes qui se déforment légèrement, pas de grille rigide
- **Shapes organiques** : Sections délimitées par des courbes, pas de lignes droites
- **Asymétrie calculée** : Images décalées, texte qui "flotte" autour d'éléments visuels
- **Espace blanc organique** : Blanc pur mais avec des formes arrondies qui le délimitent

### Signature Elements
1. **Blob shapes** : Formes organiques colorées qui délimitent les sections
2. **Illustrations custom** : Dessins minimalistes de mains, cœurs, maisons (symbolique)
3. **Gradients doux** : Transitions douces entre couleurs terreuses

### Interaction Philosophy
- Interactions qui semblent naturelles, pas "clicky"
- Hover = légère rotation + scale (1.02x)
- Animations qui suivent les courbes, pas les lignes droites
- Feedback haptique simulé par des animations fluides

### Animation
- **Entrance** : Fade-in + scale (0.95 → 1) avec easing ease-out
- **Hover** : Rotation légère (2-3°) + scale (1.05) + shadow
- **Scroll** : Éléments qui se révèlent avec une animation "organelle" (pas linéaire)
- **Blob animation** : Formes organiques qui "respirent" légèrement

### Typography System
- **Display** : Sohne (sans-serif moderne, 52-68px) pour titres = humanité moderne
- **Heading** : Sohne SemiBold (sans-serif, 32-40px) = clarté chaleureuse
- **Body** : Sohne Regular (sans-serif, 16px) = lisibilité naturelle
- **Accent** : Sohne Bold pour CTAs et labels
- **Hierarchy** : Contraste doux, pas agressif (1.5:1 ratio)

## Approche 3 : Luxe Épuré Scandinave
**Probabilité : 0.06**

### Design Movement
Luxe scandinave : minimalisme extrême, matériaux nobles (typographie fine, espacements généreux), palette neutre avec un accent de couleur précis.

### Core Principles
1. **Moins c'est plus** : Chaque pixel compte. Rien de superflu. Perfection par l'absence.
2. **Espacements généreux** : Respiration maximale entre éléments = luxe et clarté
3. **Typographie fine et élégante** : Polices légères, tracking large, leading généreux
4. **Accent unique** : Une seule couleur "riche" pour créer l'impact (or, cuivre, bleu marine)

### Color Philosophy
- **Palette primaire** : Blanc cassé #FAFAF8 (fond), noir très doux #1A1A1A (texte), or subtil #B8956A (accent)
- **Reasoning** : Minimalisme scandinave = blanc/noir. Or = luxe et chaleur. Très peu de couleur = impact maximal.
- **Emotional intent** : Sophistication, exclusivité, confiance par la retenue.

### Layout Paradigm
- **Grille 2 colonnes asymétrique** : 60/40 ou 70/30 pour créer du dynamisme
- **Espacements énormes** : 80px+ entre sections
- **Typographie comme décoration** : Les titres sont les seuls éléments "visuels"
- **Images rares et impactantes** : Peu d'images, mais très soignées, en noir & blanc ou avec accent or

### Signature Elements
1. **Ligne or fine** : Accent horizontal qui sépare les sections
2. **Typographie surdimensionnée** : Titres très grands, très espacés
3. **Minimalisme extrême** : Aucune icône, aucune illustration, juste du texte et des images

### Interaction Philosophy
- Interactions invisibles : transitions très lentes (0.5s+), presque imperceptibles
- Hover = changement de couleur très subtil (gris clair)
- Pas d'animations flashy, juste de la fluidité
- Feedback minimal mais présent

### Animation
- **Entrance** : Fade-in très lent (600ms)
- **Hover** : Changement de couleur (300ms), pas de mouvement
- **Scroll** : Parallax extrêmement subtil (vitesse 0.3x)
- **CTA** : Underline qui s'étire lentement (400ms)

### Typography System
- **Display** : Montserrat Thin (sans-serif ultra-léger, 56-72px) pour titres = luxe
- **Heading** : Montserrat Light (sans-serif, 36-44px) = élégance
- **Body** : Montserrat Regular (sans-serif, 16px) avec tracking large = lisibilité luxe
- **Accent** : Montserrat SemiBold pour CTAs = distinction
- **Hierarchy** : Contraste extrême par la taille (3:1 ratio), pas par la couleur

## Décision

**Approche sélectionnée : Minimalisme Institutionnel Chaleureux (Approche 1)**

Cette approche offre le meilleur équilibre pour les objectifs du client :
- **Confiance immédiate** : Structure institutionnelle rassurante
- **Humanité** : Détails chaleureux et authentiques
- **Clarté** : Hiérarchie typographique forte = parcours utilisateur fluide
- **Conversion** : CTA visibles et efficaces
- **Dualité** : Peut accueillir à la fois les particuliers (humanité) et les professionnels (rigueur)
- **SEO-friendly** : Structure sémantique naturelle
- **Performance** : Pas d'animations lourdes, chargement rapide

### Palette Finale
- **Primaire** : Bleu-gris #2C3E50 (confiance, rigueur)
- **Accent** : Vert doux #5FB878 (bien-être, propreté)
- **Fond** : Blanc pur #FFFFFF (clarté, sérénité)
- **Texte** : Gris foncé #333333 (lisibilité)
- **Secondaire** : Gris clair #F8F9FA (sections alternées)

### Typography
- **Display** : Playfair Display (serif, 48-64px)
- **Heading** : Poppins Bold (sans-serif, 28-36px)
- **Body** : Inter Regular (sans-serif, 16px)
- **Accent** : Poppins SemiBold (sans-serif, 14-16px)

### Éléments Distinctifs
- Ligne verticale fine bleu-gris entre sections
- Cartes de service avec coin coupé
- Icônes géométriques épurées
- Transitions fluides et prévisibles
- Images authentiques et rassurantes
